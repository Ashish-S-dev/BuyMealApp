# 🍔 BuyMeal – Food Delivery Web Application

BuyMeal is a full-stack food delivery web application that simulates a real-world online food ordering system.  
The platform allows users to browse food items, add them to a cart, and place orders through a smooth and intuitive workflow.

This project was built to strengthen my understanding of full-stack development, backend APIs, and database integration using Java and Spring Boot.

---

## 🚀 Features

- 🔐 User Authentication (Signup & Login)
- 🛒 Add food items to cart
- 💰 Dynamic cart total calculation
- 📦 Order placement system
- 📱 Responsive user interface for different screen sizes
- ⚡ Smooth and simple ordering workflow

---

## 🛠 Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Java
- Spring Boot

**Database**
- PostgreSQL

---

## ⚙️ How It Works

1. Users create an account or log in.
2. Browse available food items on the platform.
3. Add items to the cart.
4. The cart automatically updates the total amount.
5. Users can review their cart and place an order.
6. Ordered items move to the order list for processing.

## 💻 Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository

git clone https:https://github.com/Ashish-S-dev/BuyMealApp


2. Open the project in your preferred IDE (IntelliJ / Eclipse / VS Code)

3. Configure PostgreSQL database

4. Update the database configuration in `application.properties`

spring.datasource.url=jdbc:postgresql://localhost:5432/buymeal
spring.datasource.username=your_username
spring.datasource.password=your_password


5. Run the Spring Boot application
